#  시멘틱 태그



- div 태그 사용을 최소한으로 하여 HTML파일의 가독성을 최대한 올리는 방법



```html
<header>
	<nav>
    	<ul>
            <li></li>
            <li></li>
        </ul>
    </nav>
</header>

<main>
    <section>
    	<figure></figure>
        <figcaption></figcaption>
        <picture>
        	<img />
        </picture>
    </section>
    <section>
    	<article>
            <h1></h1>
        	<p>
                <span></span>
                <b></b>
                <strong></strong>
                <i></i>
                <em></em>
            </p>
        </article>
    </section>
</main>

<footer></footer>
```
